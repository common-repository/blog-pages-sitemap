Sitemap for blog pictures.

== Description ==

Create sitemap for blog pages with cron schedule. Use with Yoast SEO plugin

= Requirements =

* WordPress 5.0 or higher

== Installation ==

Unzip the blog-pages-sitemap folder to your WordPress plugins folder. It should work out of the box.
